# 🌊 OceanGuard — Front-End

> A plataforma que traz a inteligência dos satélites para a mão de quem vive do oceano.

---

## 📌 Sobre o projeto

O **OceanGuard** é uma landing page desenvolvida para a disciplina de **Front-End Design & Development** como parte da Global Solution 2026 da FIAP.

A proposta é apresentar uma plataforma de monitoramento oceânico inteligente, que une dados satelitais reais (NASA, ESA Copernicus, INPE e NOAA) com linguagem acessível — pensada para pescadores, cidadãos e pesquisadores brasileiros.

---

## ⚙️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML | Estrutura semântica da página |
| CSS | Estilização, responsividade e animações |
| JavaScript | Funcionalidades de botões |

---

## 🛰️ Fontes de dados reais

- [NASA Earthdata](https://earthdata.nasa.gov)
- [ESA Copernicus Marine Service](https://marine.copernicus.eu)
- [INPE](https://www.inpe.br)
- [NOAA Coral Reef Watch](https://coralreefwatch.noaa.gov)
